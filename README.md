
**# 🚦 Road Accident Analysis Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive **Road Accident Analysis Dashboard** developed using **Microsoft Power BI**. The dashboard analyzes road accident data to identify trends, accident severity, vehicle involvement, and geographical distribution. It helps stakeholders make data-driven decisions for improving road safety and reducing accidents.

---

## 🎯 Objectives
- Analyze overall road accident statistics.
- Monitor accident severity (Fatal, Serious, Slight).
- Identify accident trends over time.
- Analyze vehicle type involvement.
- Compare urban and rural accident occurrences.
- Support decision-making through interactive visualizations.

---

## 📊 Dashboard Features

- Monthly & Yearly Trends
- Vehicle Type Analysis
- Road Type Analysis
- Urban vs Rural Comparison
- Interactive Filters & Slicers
- Dynamic KPI Cards
- Visual Charts & Graphs



## 📈 Key Insights
- Identifies high-risk accident categories.
- Tracks accident trends over different periods.
- Highlights vehicle types involved in accidents.
- Compares accident distribution across road categories.
- Enables interactive exploration using filters.



## 🚀 How to Use
1. Download the repository.
2. Open `Road_Accident_Project.pbix` in Microsoft Power BI Desktop.
3. Refresh the dataset if required.
4. Explore the dashboard using slicers and filters.
